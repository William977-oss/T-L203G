# Example 2

### [Kennslufræði] Þessi grein fjallar um samanburð á virkri (active) og óvirkri (passive) kennsluaðferð í eðlisfræði í Harvard háskóla. Lesið a.m.k. inngang greinarinnar og skoðið súluritin í Myndum 1 og 2 (Fig. 1 og Fig. 2). Í þessum súluritum koma fyrir hugtökin TOF (Test of Learning) og FOL (Feeling of Learning). Skrifið nokkrar setningar þar sem þið reynið að útskýra hvers vegna fyrstu tvær súlurnar (með skyggðum bakgrunni) í myndunum eru svona ólíkar hinum súlunum.

#### The chart consists of two seperate graphs both displaying test results of two different methods of studying SP and RP and the stress level of participants. The stressed participants using RP (Retrieval Practice) actually managed to score higher to non-stressed participants on the second test which shows that RP not only helps with learning but also helps with memory performance under stress.
